# HW-OPEN-010 — Artin Prime Tower Transition Problem

Status: open-problem / finite-to-asymptotic transition surface.  
Claim class: open analytic transition problem.  
Lane: prime/operator lane, Richter-window / digit-cycle / Artin-prime cancellation surface.  
Depends on: `HW-PRIME-WEIL-006`, `HW-PRIME-WEIL-007`.

## Purpose

`HW-PRIME-WEIL-007` proves the finite orbit-classification theorem for base `10` modulo primes.

The strongest cancellation occurs at Category 3 primes:

```text
ord_p(10) = p - 1.
```

At those primes, base `10` generates the full multiplicative group `(Z/pZ)^x`, and every nontrivial multiplicative character cancels over a complete digit orbit.

`HW-OPEN-010` asks whether this finite tower of Artin-prime cancellation anchors can constrain actual Richter-window prime sums strongly enough to affect the square-root barrier.

## Transition question

Let `p` be a Category 3 prime for base `10`. Then:

```text
<10> = (Z/pZ)^x.
```

For every nontrivial character `chi mod p`:

```text
sum_{j=0}^{p-2} chi(10^j) = 0.
```

The finite cancellation is exact.

The transition problem is to determine whether this digit-cycle cancellation controls actual prime-window sums:

```text
psi_{W_K}(chi) = sum_{q in W_K} log(q) chi(q)
```

at resonant depths or across depth intervals.

## Resonant anchor question

The natural resonant depth attached to a Category 3 prime is:

```text
ell = p - 1.
```

Question:

```text
Does full digit-cycle cancellation at period ell force improved bounds for psi_{W_ell}(chi) beyond generic sieve-scale control?
```

A positive result requires connecting prime occurrence in the value window to the digit-cycle compartments generated by powers of `10` modulo `p`.

The finite orbit theorem alone does not do this.

## Tower question

Let:

```text
p_1 < p_2 < p_3 < ...
```

be primes for which `10` is a primitive root.

Open:

```text
Do the resonant depths ell_i = p_i - 1 provide enough cancellation anchors to constrain Richter-window variance between anchors?
```

This is the finite-to-asymptotic transition problem.

## Artin connection

Artin's primitive root conjecture predicts that base `10` is a primitive root modulo infinitely many primes, with positive density subject to the usual local exclusions.

Hooley proved the conjecture conditionally on GRH for suitable Dedekind zeta functions.

In this lane, the finite relevance is:

```text
Artin primes for base 10 are exactly the Category 3 full-orbit primes of HW-PRIME-WEIL-007.
```

No claim is made here that Artin's conjecture is proved unconditionally.

## What would constitute progress

Progress would be any theorem that transfers finite digit-cycle cancellation into actual prime-window control. Candidate forms:

1. a resonant-depth bound for Category 3 primes;
2. an interpolation principle between Artin-prime resonant depths;
3. a density-weighted variance decomposition using the tower of Category 3 primes;
4. a bound showing that non-Artin partial or degenerate orbit components are negligible in the full Richter variance;
5. a proof that the Artin-prime cancellation anchors force a sub-Siegel-Walfisz variance scale for fixed primorial packets.

The strongest target would be an unconditional critical-scale variance bound. That remains GRH-strength by `HW-PRIME-WEIL-005`.

## Non-claims
