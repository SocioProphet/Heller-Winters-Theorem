## Claim boundary

Prime/operator-lane content uses the claim classes from `claim-ledger-v0.1.md`: `C0`, `C1`, `C2`, `C3`, `D0`, and `E0`.

The prime-circle correspondence note is not a physical claim, not an RH claim, and not an asymptotic prime-gap theorem. Its current claim surface is:

- `C0`: definitions and correspondence typing;
- `C1`: bounded finite verification;
- `C3 imported background`: standard number-theoretic facts when explicitly cited.

Reserved identifiers remain open and cannot be cited as active claims.
