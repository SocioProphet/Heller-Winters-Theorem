# HW-PRIME-WEIL-014 — Non-Cancelling Fraction Theorem

Status: finite arithmetic theorem surface.  
Claim class: theorem-grade finite character-count identity; diagnostic implications only.  
Lane: prime/operator lane, orbit-class / primorial packet surface.  
Depends on: `HW-PRIME-WEIL-007`, `HW-PRIME-WEIL-012`, `HW-PRIME-WEIL-013`.

## Purpose

This document records the exact finite theorem controlling how many characters fail to cancel on the base-10 digit orbit.

It has two forms:

1. a local prime-modulus theorem;
2. a primorial diagonal-orbit theorem.

Both are unconditional finite group-character identities. Neither proves RH, GRH, Artin's conjecture, or any variance bound.

## Local theorem

Let `p` be a prime with `gcd(p,10)=1`. Let:

```text
G_p = (Z/pZ)^x,
H_p = <10> subset G_p,
d_p = ord_p(10).
```

Then:

```text
|G_p| = p-1,
|H_p| = d_p,
[G_p : H_p] = (p-1) / d_p.
```

A character `chi in hat(G_p)` cancels over the base-10 orbit iff its restriction to `H_p` is nontrivial:

```text
sum_{h in H_p} chi(h) = 0 iff chi|_{H_p} is nontrivial.
```

A character is non-cancelling iff it is trivial on `H_p`.

Therefore the number of nontrivial non-cancelling characters is:

```text
n_nc(p) = (p-1) / ord_p(10) - 1.
```

## Proof

Characters of `G_p` that are trivial on `H_p` are exactly characters of the quotient group:

```text
G_p / H_p.
```

The quotient has size:

```text
|G_p/H_p| = (p-1) / ord_p(10).
```

The trivial character is one of these quotient characters. Removing it gives:

```text
n_nc(p) = |G_p/H_p| - 1 = (p-1)/ord_p(10) - 1.
```

This is finite character orthogonality.

## Artin-prime corollary

If `10` is a primitive root modulo `p`, then:

```text
ord_p(10) = p-1.
```

Therefore:

```text
n_nc(p) = (p-1)/(p-1) - 1 = 0.
```

Thus base-10 Artin primes have no local nontrivial non-cancelling characters.

This is an exact finite corollary. It does not prove Artin's conjecture or assert infinitely many such primes.

## Primorial diagonal-orbit theorem

Let:

```text
P = product_{p in S} p
```

where `S` is a finite set of primes and `P` is coprime to `10` after removing terminating factors as appropriate in the character packet model.

Let:

```text
G_P = (Z/PZ)^x ~= product_{p in S} (Z/pZ)^x.
```

Let `H_P` be the diagonal base-10 orbit generated by `10` inside `G_P`:

```text
H_P = <10> subset G_P.
```

Its order is:

```text
|H_P| = lcm_{p in S, gcd(p,10)=1} ord_p(10).
```

The number of nontrivial characters of `G_P` that are trivial on `H_P` is:

```text
N_nc(P) = phi(P) / |H_P| - 1
        = phi(P) / lcm_p ord_p(10) - 1.
```

This is the simultaneous diagonal-orbit version of the local theorem.

## Interpretation

The local theorem says that orbit quality is exactly measured by the quotient size:

```text
|G_p/H_p| = (p-1)/ord_p(10).
```

Full-orbit primes have quotient size `1`, hence no non-cancelling local characters.

Partial-orbit primes have quotient size greater than `1`, hence some non-cancelling local characters.

The primorial theorem says that simultaneous resonance is governed by the diagonal orbit size, not by a simple sum of local non-cancelling counts.

## Boundary

This theorem counts characters. It does not bound their prime-window character sums.

A count of non-cancelling characters is not a variance estimate unless paired with analytic control of the corresponding `psi_{W_K}(chi)` values.
