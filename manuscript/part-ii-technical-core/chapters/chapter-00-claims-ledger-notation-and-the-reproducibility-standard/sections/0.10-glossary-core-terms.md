## Canonical Terms

**Log-circle phase map**: The map p ↦ θ(p) = (ln p mod 2π) embedding primes onto the unit circle via logarithmic phase.

**Traversal index k(p)**: The integer k(p) = floor(ln p / 2π), counting how many full revolutions ln p has completed.

**Identity-coupled feature map**: A feature representation in which the coordinate system is fixed by the mathematical object's identity (prime index or value), not by exogenous tuning.

**Wheel sieve generator**: The periodic sequence of wheel-admissible residues mod M = p_1·p_2·…·p_r that generates candidate positions for primes without redundant multiples.

**Certified window + sieve policy layer**: The combination of a bounded prime window [x, x+H] with a deterministic elimination policy, yielding a certified set of prime candidates.

**Ablation-first validation protocol**: The experimental discipline of testing each component of a model in isolation before combining, to attribute effects unambiguously.

**Arithmetic null**: The set of wheel-admissible composites (i.e., integers n with gcd(n, M) = 1 that are not prime); excludes primes from the null distribution.

**Wheel-admissible residue**: A residue class r mod M satisfying gcd(r, M) = 1, meaning integers in this class are not eliminated by the base primes p_1, …, p_r dividing M.

**Sieve policy layer**: A deterministic constraint rule-set together with an audit ledger that records why each candidate was retained or eliminated.

**Certified prime candidate**: An integer that survives all applicable policy-layer elimination rules within a bounded window; not asserted to be prime until verified.

**Mean resultant length R**: The modulus of the vector mean of unit-circle points; R ∈ [0, 1] with R ≈ 1/√N expected under uniform-phase null (see §Issue 7 note).

**Kuiper statistic V**: The sum of the maximum positive and maximum negative deviations of the empirical CDF from the theoretical CDF on the circle; invariant under cyclic shift.

**Watson statistic U²**: A rotation-invariant circular analogue of the Cramér–von Mises statistic; preferred over linear KS for phase data on the unit circle.

**Rayleigh test statistic z**: z = N · R²; tests whether phases cluster significantly away from the uniform distribution on the circle.

**Wheel modulus M**: The product M = p_1 · p_2 · … · p_r of the first r primes; defines the periodicity of the wheel sieve generator.

**Default wheel order Q**: The default choice Q = 7, giving M = 2·3·5·7·11·13·17 = 510510; sensitivity tables should cover Q ∈ {5, 7, 11}.
