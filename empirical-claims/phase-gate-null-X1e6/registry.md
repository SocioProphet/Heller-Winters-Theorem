## Circular-distance convention

All circular values live in `[0,1)`.

The one-sample Kuiper statistic against the uniform distribution is computed from sorted phases:

```math
D^+ = \max_{1 \le i \le N}\left(\frac{i}{N} - x_i\right)
```

```math
D^- = \max_{1 \le i \le N}\left(x_i - \frac{i-1}{N}\right)
```

```math
V = D^+ + D^-
```

## Primary statistic

```math
T(S) = \max_{\omega \in \Omega} \sqrt{|S|}\,V_\omega(S)
```

where `V_omega(S)` is the Kuiper statistic of `{theta_omega(n): n in S}`.
