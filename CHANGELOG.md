# Changelog

## 2026-05-14 — Umbrella reframing

The repository README was reframed so the canonical object is the **Heller–Winters Programme**, not a completed Heller–Winters Theorem and not only a prime/operator proof attempt.

This is a semantic scope change, not a theorem promotion. The repository now houses an umbrella programme whose primary lanes are:

1. gravitational information bounds;
2. octonionic Hopf geometry;
3. spectral zeta regularization;
4. descent/topos semantics;
5. information-geometric learning dynamics.

Prime/operator work remains active, but it is treated as a subordinate lane inside the wider programme.

### Claim boundary

The phrase **Heller–Winters Theorem** remains reserved for future theorem-worthy results only after they are formally stated and proved. Current work is programme-level, definitional, empirical, conjectural, or theorem-candidate work unless separately promoted under the repository's claim discipline.

### Downstream implication

Existing prime/operator claim-ledger infrastructure remains lane-scoped unless and until a programme-wide claim-ledger extension is explicitly introduced. Other SocioProphet repositories that use Heller methodology are not automatically inside this programme; they require explicit cross-repository lane declarations.
